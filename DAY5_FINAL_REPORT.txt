    ├─ Install production deps

    └─ Provide HTTPS domain
        ↓

    ├─ React/Next.js UI

    ├─ Shopping cart
    ├─ Checkout flow
    └─ Payment processing
        ↓
    Backend API Routes

TROUBLESHOOTING REFERENCE
════════════════════════════════════════════════════════════════════════════════

Problem: "Repository not found" when pushing to GitHub
  Solution: Create repository first at https://github.com/new

Problem: Build fails in GitHub Actions
  Solution: Check GitHub Secrets are complete and correctly named
            Review Actions logs for specific build error

Problem: Railway deployment fails
  Solution: Check environment variables spelling and values
            Review Railway logs for build/runtime errors
            Verify .env file not committed to repo

Problem: Webhooks not firing
  Solution: Verify webhook URLs are correct
            Check Railway domain is accessible
            Test webhook manually in provider dashboard

Problem: CORS errors in browser
  Solution: Check API route CORS headers
            Verify NEXT_PUBLIC_API_URL matches Railway domain
            Review browser console for specific errors
